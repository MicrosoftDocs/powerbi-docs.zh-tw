# <a name="what-to-do-if-sign-up-is-disabled"></a>如果已停用註冊，該怎麼辦

您已嘗試註冊 Power BI，並收到註冊已停用的訊息。 基於各種原因，某些組織會封鎖成員進行自助式註冊的能力。  例如，您的組織可能有要求所有授權 (即便是免費授權) 和訂用帳戶都由集中式的 IT 部門或技術支援中心來管理的原則。
