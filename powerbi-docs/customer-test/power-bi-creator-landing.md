# <a name="create-with-power-bi"></a>使用 Power BI 建立

您是 Power BI 建立者嗎？ 您連線至資料、設計報表以及建立儀表板嗎？  您是建立自己應用程式的開發人員嗎？ 您管理 Power BI 嗎？ 這些資源將可協助您快速上手。 歡迎使用！

您其實是 Power BI 取用者嗎？ 也歡迎您使用！ 建議您從[適用於取用者的 Power BI 開始](consumer/power-bi-consumer-overview.md)。
