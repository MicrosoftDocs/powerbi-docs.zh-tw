# <a name="copy-and-paste-a-report-visualization"></a>複製並貼上報表視覺效果
複製並貼上視覺效果時，需要有報告的編輯權限。 在 Power BI 服務中，這表示在[編輯檢視](../consumer/end-user-reading-view.md)中開啟報告。 無法複製並貼上儀表板上的視覺效果。

1. 開啟至少有一個視覺效果的報告。  

2. 選取視覺效果並使用 **Ctrl+C** 複製，並使用 **Ctrl+V** 貼上。
