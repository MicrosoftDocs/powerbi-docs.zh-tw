Power BI 會將您的報表、儀表板和磚保留在工作區中。 若要內嵌這些項目，您必須建立這些項目，並將其上傳至工作區。

>[!TIP]
>如果您已經有工作區，就可以略過此步驟。

若要建立工作區，請執行下列動作：

1. 登入 Power BI。

2. 選取 [工作區]。

3. 選取 [建立工作區]。

4. 為工作區命名，然後選取 [儲存]。
