1. 登入 [Azure 入口網站](https://portal.azure.com)。

2. 選取 **應用程式註冊**。 如果您看不到此選項，請進行搜尋。

3. 開啟您在 [步驟 1-註冊 Azure AD 應用程式](#step-1---register-an-azure-ad-application)中建立的 Azure AD 應用程式。

4. 從 [ *管理* ] 功能表選取 [ **驗證**]。
