若要取得工作區識別碼 GUID，請遵循下列步驟：

1. 登入 Power BI 服務。

2. 開啟您想要內嵌的報表。

3. 複製 URL 中的 GUID。 GUID 是 **/groups/** 與 **/reports/** 之間的數字。
