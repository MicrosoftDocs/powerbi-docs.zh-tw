# <a name="tips-to-sort-and-distribute-data-plots-in-power-bi-reports"></a>排序和散發 Power BI 報表中資料繪圖的提示

本文適用於設計 Power BI 報表，且正在使用資料繪圖視覺效果的報表作者。

請觀看影片，其示範排序和散發 Power BI 報表中資料繪圖的前九個提示。

<iframe width="560" height="315" src="https://www.youtube.com/embed/BnhA4qa-9Jk" frameborder="0" allowfullscreen></iframe>

## <a name="tips"></a>祕訣

總結而言，排序和散發 Power BI 報表中資料繪圖的提示包括：

1. 排序名義類別
1. 排序間隔類別
1. 排序序數類別
1. 視覺化距離較遠的繪圖
1. 視覺化擁擠的類別繪圖
1. 視覺化擁擠的時間繪圖
1. 散發多個維度
1. 分類資料繪圖
1. 區分值繪圖
